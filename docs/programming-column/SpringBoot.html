常用条件注解​

注解含义实现备注
@ConditionalOnClass判断类是否存在@Conditional(OnClassCondition.class) 通过ClassLoader去加载类反义词 @ConditionalOnMissingClass
@ConditionalOnBean判断Bean是否存在@Conditional(OnBeanCondition.class) 通过BeanFactory查询Bean反义词 @ConditionalOnMissingBean
@ConditionalOnSingleCandidate判断某个类型的Bean是否只有一个，或者有多个并且有一个是Primary@Conditional(OnBeanCondition.class)
@ConditionalOnProperty判断属性配置@Conditional(OnPropertyCondition.class) 通过Environment查询属性
@ConditionalOnWebApplication判断是不是web应用@Conditional(OnWebApplicationCondition.class)反义词 @ConditionalOnNotWebApplication
